# 10 - 内容禁忌

## 【必须避免】

### 通用禁忌：

1. **概念化表达**：不要说"改革开放带来了巨大变化"，要写具体的变化
2. **脸谱化人物**：没有纯粹的坏人或好人
3. **廉价的和解**：不要让所有矛盾都圆满解决
4. **过度的象征**：意象要自然，不要刻意
5. **回避性与欲望**：这是故事内核，但要写得克制隐晦
6. **政治化处理**：写人性，不要写政策对错

### 倒叙线特定禁忌：

7. **知识分子腔调**："我"是从村里出来的孩子，不要用成年视角说教
8. **过度美化乡村**：贫穷就是贫穷，不要浪漫化
9. **过度丑化乡村**：也有温情和尊严
10. **编造风俗**：不要编造不存在的河南农村风俗

### 顺叙线特定禁忌：

11. **程序员刻板印象**：不要写成"码农""加班狗"的扁平形象
12. **中产阶级焦虑的贩卖**：要真实，不要贩卖焦虑和怀旧
13. **成功学叙事**：这不是励志故事，是关于妥协的故事
14. **过度的技术描写**：技术是背景，不是主角
15. **城市vs乡村的简单对立**：两者都有复杂性

---

## 【尺度把握】

### 性：
- 可以暗示，不要直写
- 用环境、细节间接表现
- 例：村里的流言、暧昧的眼神、关闭的房门

### 暴力：
- 可以写死亡，但不渲染血腥
- 例：粮站塌方、煤矿事故、水灾
- 重点是后果和影响，不是过程

### 道德：
- 不要站在上帝视角评判
- 让读者自己判断对错
- 每个人都有其正当性和局限性

### 职场政治：
- 可以写，但不要写成宫斗剧
- 聚焦在"我"的体验和选择上
- 不要过度渲染阴谋和算计

### 婚姻矛盾：
- 真实但克制，不要写成家庭伦理剧
- 聚焦在内心的挣扎和妥协上
- 不要渲染争吵和冲突

## 【示例对比】

### 概念化 vs 具体化

```
✗ "改革开放给农村带来了巨大变化。"

✓ "包产到户那年，粮站开始废弃。王屠户买了台电视机，
   全村人都去他家看。后来三叔去了山西煤矿，再也没回来。"
```

### 脸谱化 vs 复杂化

```
✗ "王屠户是个坏人，总是欺负孙寡妇。"

✓ "王屠户去帮孙寡妇种地。天黑了，他扛着锄头下山，
   孙寡妇跟在后面。村里人看见了。"
```

### 廉价和解 vs 真实结局

```
✗ "最后我和父亲和解了，他终于理解了我，我们拥抱在一起。"

✓ "父亲在电话里说：'中，你在郑州好好干。'
   我说：'嗯。'电话那头沉默了很久。"
```

### 知识分子腔调 vs 乡土语言

```
✗ "我深刻地意识到，贫困是一切苦难的根源，
   而教育是改变命运的唯一途径。"

✓ "我爹说：'考上大学，就不用挖土了。'"
```

### 过度美化 vs 真实刻画

```
✗ "乡村的生活虽然贫困，但充满了温情和诗意。"

✓ "腊月二十三那天，家里只剩半口袋红薯干。
   我娘说：'等你爹回来，能买点白面。'"
```

## 【检查清单】

写完一段后，自查：

- [ ] 是否有概念化表达？
- [ ] 是否将人物脸谱化？
- [ ] 是否过度抒情或说教？
- [ ] 是否符合人物的身份和年龄？
- [ ] 是否保持了冷静克制？
- [ ] 是否贩卖焦虑或怀旧？
- [ ] 是否简单化城乡对立？
