# 05 - 乡土细节

这是倒叙线的基础，必须真实、具体、有质感。

## 【地理与自然】

- **伏牛山**：土石山，不高，但绵延；黄土与石头混杂
- **伊河故道**：经常断流，发大水时又很凶猛；河床宽阔，淤泥深厚
- **黄土**：赭色的，干裂的，粉尘很大；风一吹满天土
- **季节特征**：
  - 冬天的雪：大雪封山，窑洞冒烟
  - 夏天的伏天：闷热难耐，知了叫个不停
  - 秋天的麦收：金黄的麦浪，打谷场上忙碌

---

## 【村落结构】

- **窑洞**：拱形的门、土坯的墙、煤油灯；冬暖夏凉，但潮湿压抑
- **打谷场**：村里的公共空间，开会、放电影、晒粮食
- **粮站**：集体经济时代的遗迹，后来慢慢废弃
- **供销社**：卖火柴、煤油、化肥的地方，后来也荒废了
- **碾盘**：石头做的，村口的聚集地，老人们聊天的地方
- **水井**：村里的生命线，早晚打水的女人们
- **村口的大树**：古槐或老榆树，树下的石碾子

---

## 【生产生活】

### 农作物
- **主粮**：小麦、玉米、红薯
- **蔬菜**：土豆、白菜、萝卜（窖藏过冬）
- **经济作物**：核桃、柿子（后来种的）

### 农具
- **扁担**：挑水、挑粪、挑一切
- **锄头**：刨地、除草
- **镰刀**：割麦子、割草
- **箩筐**：竹编的，装粮食、装菜
- **铁锹**：挖地、挖渠
- **木犁**：牛拉的，翻地用

### 牲畜
- **牛**：耕地用，一家最值钱的财产
- **猪**：养一年，过年杀，一年的油水
- **鸡**：散养的，下蛋补贴家用
- **驴**：拉磨的，后来越来越少

### 饮食
- **日常**：红薯干、玉米糊、糠窝窝（贫困时）
- **改善**：白面馍、烩面（逢年过节）
- **特色**：胡辣汤、羊肉汤、烩面
- **腌制**：腌白菜、腌萝卜（过冬）

### 衣物
- **的确良衬衫**：80年代的时髦
- **补丁裤子**：新三年旧三年，缝缝补补又三年
- **解放鞋**：黑色胶鞋，结实耐穿
- **千层底布鞋**：母亲纳的，一针一线
- **棉袄**：冬天的必需品，厚重

---

## 【家庭关系】

### 父权制
- 父亲在家里的绝对权威
- 儿子不能顶嘴，女儿更没有话语权
- 家族大事，父亲说了算
- 打骂孩子天经地义

### 母亲角色
- 操持家务：做饭、喂猪、种菜、纳鞋底
- 管孩子：照顾、教育、但没有决定权
- 忍辱负重：在婆家、在丈夫面前
- 精打细算：省吃俭用，供孩子读书

### 兄弟关系
- 分家：父母在不分家，父母去世后分
- 财产：地、房、老物件的分配
- 赡养老人：兄弟轮流，但矛盾重重
- 攀比：谁家日子过得好，谁家孩子争气

### 婆媳关系
- 永恒的张力
- 婆婆对媳妇的挑剔
- 媳妇的隐忍和反抗
- 儿子夹在中间的两难

### 重男轻女
- 刻在骨子里的观念
- 女儿：赔钱货，早晚是别人家的
- 儿子：传宗接代，光宗耀祖
- 生不出儿子，媳妇抬不起头

---

## 【村庄伦理】

### 面子
- 比生存更重要的东西
- 宁可自己受苦，不能被人看不起
- 婚丧嫁娶要体面，借钱也要办
- 穷不能穷志，不能让人瞧不起

### 流言
- 村里最快的传播途径
- 好事不出门，坏事传千里
- 女人们聚在一起，嘴就停不下来
- 流言能杀人，比刀子还快

### 宗族
- 姓氏：同姓是一家，不能结婚
- 祠堂：祭祖的地方，族规的象征
- 祖坟：风水宝地，不能乱动
- 族长：村里的权威，调解纠纷

### 婚丧嫁娶
- 巨大的经济压力
- 村里最重要的社交场合
- 攀比的战场：谁家办得体面
- 礼金：人情债，世代传承

### 攀比
- 贫穷中的虚荣
- 谁家盖了新房，谁家买了电视
- 谁家儿子考上大学，谁家女儿嫁得好
- 比来比去，永无休止

## 【重要原则】

1. **不要编造不存在的风俗**
2. **不要过度民俗化**（不写成民俗志）
3. **方言要适度**，不影响阅读
4. **乡土细节要自然融入叙事**，不要说明性描写

### 在场景中自然融入
```
✓ "母亲在院子里晒红薯干，用竹篾席摊开，一层一层。"
✗ "红薯干是河南农村的传统食物，制作方法是……"
```

### 通过对话展现方言
```
✓ "中不中？"父亲问。
   "中！"我说。
✗ "中不中"是河南方言，意思是"行不行"。
```

### 通过冲突展现伦理
```
✓ 婆婆说："女娃儿读那么多书干啥？早晚是人家的人。"
✗ 在农村，重男轻女的观念很严重……
```

## 【农村习俗与特殊现象】

### 过年习俗
- 擂社：春节期间各村组织的打鼓比赛，年轻人抬着大鼓走街串巷，鼓声震天
- 唱大戏：请豫剧团来村里唱戏，连唱三天，全村人搬着小板凳去看
- 社火表演：舞龙舞狮、踩高跷、划旱船等传统表演
- 祭祖：大年初一全家到祖坟上香烧纸，祈求祖先保佑

### 特殊社会现象
- 盗墓：80-90年代洛阳地区盗墓猖獗，很多农民参与，与贫困有关
- 偷火车：90年代陇海铁路沿线村民偷盗火车物资的现象
- 赌博：农村常见的麻将、推牌九等赌博活动，过年期间尤其盛行

### 重大历史事件
- 1948年洛阳战役：解放战争中的重要战役
- 1958年大跃进：洛阳地区大炼钢铁运动
- 1960-1962年三年困难时期：饥荒对农村的影响
- 1980年代包产到户：农村经济体制改革
- 1990年代打工潮：农村劳动力外出务工
