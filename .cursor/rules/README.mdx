## 💡 核心原则（记住这7句话）

1. **写什么人们做了什么，不写人们感受到什么**
2. **让读者哭，不要让人物哭**
3. **越是残酷的事，越要用平静的语言**
4. **最重要的话，永远说不出口**
5. **好的文学不是讲故事，是在读者心里种下种子**
6. **每个人物都是时代的隐喻**
7. **写出一部50年后仍会被阅读的作品**
