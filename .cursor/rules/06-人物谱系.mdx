# 06 - 人物谱系（家族树 × 弧线模板）

## 【家族树与关系网】

```
石家（我｜石头，1979.10.14，射手座）
├─ 父亲（沉默/固执/以尊严为底线）
├─ 母亲（操持/计较/以生存为本能）
├─ 哥哥（农民，留守乡村）
├─ 大妹（1985年生，大学毕业，北京成家）
└─ 二妹（远嫁新疆）

村落关系（倒叙线）
├─ 老支书（秩序的守望者）
├─ 顺子（为爱冒险者）— 穗子（在贫穷与体面之间）
├─ 孙寡妇（被流言包围的生存者）— 王屠户（善意被误读）
├─ 林晓（外来启蒙者）
├─ 农村喜欢的女孩（少年的朦胧情感）
└─ 老家混的同学（留守vs出走的对照）

深圳关系网（顺叙线核心）
├─ 两个铁哥们（从老家带出的兄弟）
├─ 四个大哥（罩着我的人，教我江湖规矩）
├─ 第一个老板（学徒对象）
├─ 第一批员工（信任我的人）
└─ 关键客户（第一桶金的来源）

情感线索
├─ 农村女孩（少年时的朦胧）
├─ 洛阳高中同学（很喜欢我，代表另一种可能）
└─ 妻子（2008年29岁结婚，陪我辗转各城市）
```

→ 详尽人物素材参阅《06-人物图谱》（旧）。本文件提供结构与方法。

---

## 【原型库（可重组）】

- 守旧者：老支书/父亲（守住体面与秩序）
- 冒险者：三叔/顺子（流动性与风险）
- 投机者：小生意人（抓缝隙的本领）
- 学而变者：我/林晓（知识带来的断裂）
- 溃败者：孙寡妇（被时代碾过的人）

使用法：每个原型至少赋予一处“自洽的矛盾”（例如：暴躁与保护、刻薄与关怀）。

---

## 【人物弧线模板】

1) 成长弧：缺失 → 追求 → 试错 → 选择 → 付账 → 局部修复
2) 和解弧：误解 → 推远 → 对照 → 松动 → 对话/沉默 → 部分理解
3) 溃败弧：渴望 → 受阻 → 偏门 → 越界 → 代价 → 沉落/余烬
4) 守望弧：看见 → 坚守 → 被误解 → 持续 → 迟来的回声

### 弧线填空卡（示例）
```
人物：父亲
缺失/渴望：体面与被看见
选择：拒绝借礼金，宁可简单办
代价：被族人嘲讽，与亲戚决裂
回声：多年后我在城市里学会了“沉默的体面”
```

---

## 【张力网络（关系即冲突）】

- 纵轴：父辈—我—子辈（代际）
- 横轴：乡村—城市（空间）
- 斜轴：传统—现代（价值）

在章内至少激活两轴冲突，例如：
- 父亲的体面（传统/乡村） × 我在北京的窘迫（现代/城市）

---

## 【人物出场规则】

1. 用一个动作或物件定义（而非履历）。
2. 初登场不解释其“好坏”，用行为与他人反应呈现。
3. 每次再出场，带来一个新的“代价/回声”。
