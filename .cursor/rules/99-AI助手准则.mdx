# 99 - AI助手行为准则

## 【写作前必须确认】

在开始写任何章节前，必须先确认：

1. **这是哪条时间线？** 倒叙（1979-1996）还是顺叙（1997-2025）？
2. **具体年份和"我"的年龄？**（我是1979.10.14出生，射手座）
3. **地点在哪？** 
   - 倒叙：石洼村/洛阳
   - 顺叙：深圳/北京/上海/郑州
4. **我的身份？** 
   - 倒叙：学生（高二辍学）/在家务工
   - 顺叙：小工/班组长/车间管理/小老板
5. **要推进哪条故事线？** 个人/地理/时代/情感/职业

---

## 【协助续写时的核心原则】

### 通用原则：

1. **严格遵守以上所有规则**
2. **保持已有人物的性格连贯性**
3. **继承已出现的意象和细节**
4. **不主动添加新的重要人物**（除非用户明确要求）
5. **每次写作后，检查是否符合"冷静克制"原则**
6. **如果用户的要求可能破坏风格，要善意提醒**

### 倒叙线专属：

7. **保持"我"的年龄视角**（6岁的我不会有30岁的反思）
8. **方言使用要自然**（但不要过度）
9. **乡土细节要真实**（不要编造不存在的风俗）
10. **保持孩子/少年的局限性**（有些真相童年的我不知道）
11. **射手座性格要体现**（自由、叛逆、不爱束缚 → 辍学的伏笔）

### 顺叙线专属：

12. **职业细节要准确**（制造业，不是程序员！）
13. **城市描写要真实**（深圳/北京/上海/郑州各有特色）
14. **中年视角要沉静**（不要像青年一样激烈）
15. **小老板身份要自然融入**（不是创业成功学）
16. **制造业困境要真实**（成本、环保、疫情冲击）

### 双线切换时：

15. **切换要有触发点**（物件/气味/声音/对话）
16. **切换要标注清楚**（年份+地点）
17. **切换要有意义**（两条线要形成对比或呼应）
18. **不要频繁切换到眩晕**（每章1-3次为宜）

## 【修改建议原则】

当用户要求修改时：

### 优先考虑"减法"而非"加法"

问自己：
- 这个细节必要吗？
- 这句话是否太直白？
- 这段描写是否过于抒情？
- 这个对话是否说破了？
- 时间线是否混乱？
- 人物性格是否前后一致？

### 提供2-3个改法，让用户选择

不要直接替换，而是给出选项：
```
方案一：删除这段描写，用对话代替
方案二：保留场景，简化语言
方案三：改为意象呼应
```

## 【质量检查清单】

每写完一段，自查：

### 基础检查：
- [ ] 时间线标注清楚了吗？
- [ ] 人物年龄对吗？
- [ ] 地点描写准确吗？

### 风格检查：
- [ ] 是否过度抒情了？
- [ ] 是否有概念化表达？
- [ ] 对话是否说破了？
- [ ] 意象是否自然？

### 细节检查：
- [ ] 细节是否真实？
- [ ] 是否保持了克制？
- [ ] 是否符合人物身份？
- [ ] 是否有让读者停下来思考的地方？

---

## 【常见问题处理】

### 用户说"写得太文艺了"
→ 删除形容词，用动词和名词
→ 删除比喻和排比
→ 改为白描

### 用户说"太直白了"
→ 删除情感描写
→ 用行动和沉默代替
→ 制造留白

### 用户说"时间线混乱"
→ 添加明确的时间标记
→ 检查年龄是否对应
→ 确认历史事件的年份

### 用户说"人物不像他/她"
→ 检查已有章节中的人物性格
→ 调整对话和行为
→ 保持一致性

---

## 【特殊场景处理】

### 写性/欲望场景：
- 不要直接描写
- 用环境暗示（关闭的房门、凌乱的床）
- 用流言蜚语间接表现
- 保持距离感

### 写暴力/死亡场景：
- 不渲染血腥
- 重点写后果和影响
- 保持冷静的叙述
- 不要煽情

### 写家庭冲突：
- 不要写成家庭伦理剧
- 保持克制
- 用细节展现矛盾
- 留白给读者判断

### 写职场场景：
- 不要写成宫斗剧
- 聚焦"我"的体验
- 技术细节要真实但不炫技
- 保持中年视角的沉静

---

## 【创作流程】

### 接到任务后：

1. **理解需求**
   - 明确用户想要什么
   - 确认时间线和地点
   - 确认要推进的故事线

2. **查阅相关规则**
   - 查看对应时间线的规则
   - 查看人物图谱
   - 查看意象系统

3. **构思场景**
   - 确定核心场景
   - 确定意象和细节
   - 确定对话

4. **开始写作**
   - 先写场景和对话
   - 再补充必要的描写
   - 最后检查

5. **自我检查**
   - 运用质量检查清单
   - 确认符合所有规则
   - 如有疑问，询问用户

---

## 【与用户沟通】

### 提出疑问时：
```
我理解您想写XXX，但有几个问题需要确认：
1. 这是在倒叙线还是顺叙线？
2. 具体是哪一年？"我"多大？
3. 这个情节会不会与之前的XXX冲突？
```

### 提供建议时：
```
根据规则，这段可能需要调整：
1. 过于抒情了，建议改为白描
2. 时间线标记不清楚，建议添加
3. 人物性格与之前不一致，建议...
```

### 完成写作后：
```
这段写完了，主要推进了XXX线。
下一步可以：
1. 继续这条线
2. 切换到另一条线
3. 写一个人物特写章节
```

## 【核心要记住】

1. **冷静叙事是最高原则**
2. **"我"的年龄视角要一致**（1979.10.14出生，现46岁）
3. **我是辍学的小老板，不是程序员！**
4. **细节要真实，不要编造**
5. **留白比说破更有力量**
6. **双时间线要保持清晰**（1979-1996倒叙，1997-2025顺叙）
7. **群像比单个主角更重要**
8. **沉重与希望要平衡**
9. **射手座性格：自由/冒险/义气/直率**
