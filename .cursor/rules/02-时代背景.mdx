# 02 - 时代背景

## 【基本信息】

- **时间跨度**：1979-2025（46年）
- **地理跨度**：河南洛阳伏牛山下→各大城市→回归郑州
- **核心地域**：石洼村（虚构），伊河故道，伏牛山区

---

## 【历史事件背景】

历史事件作为时间锚点，帮助定位故事发生的具体时代背景。

### 倒叙线时代背景（1979-2009）

#### 1978-1982：改革开放初期

- 1978年：十一届三中全会，改革开放启动
- 1979-1980：知青返城，上山下乡结束
- 1980-1982：包产到户，家庭联产承包责任制
- 集体经济解体，粮站、供销社的衰落
- 村里开始有"万元户"

**社会氛围：**
- 从集体到个人的转变
- 人们对未来充满期待和不安
- 传统秩序开始松动

#### 1983-1992：乡村巨变期

- 计划生育政策严格执行
- 外出打工潮开始（山西煤矿、广东工厂）
- 电视机、自行车、缝纫机进入农村
- 乡镇企业兴起又衰落
- 读书改变命运的信念达到顶峰

**社会氛围：**
- 乡村开始空心化
- 留守老人和儿童出现
- 城乡差距初现
- 教育热潮

#### 1993-1999：城乡差距扩大

- 1992年：邓小平南巡讲话
- 市场经济体制确立
- 农民工大潮涌入城市
- 教育产业化，农村孩子上学越来越贵
- 乡村空心化加剧，留守老人和儿童

**社会氛围：**
- 下海经商热潮
- "万元户"变成"老板"
- 农村与城市的鸿沟拉大
- 读书不一定改变命运的声音出现

#### 2000-2009：互联网时代初期

- 2000年左右：互联网开始普及
- 程序员成为热门职业
- 北上广深房价开始飙升
- 农村土地被征用，拆迁户出现
- 城乡二元结构固化

**社会氛围：**
- 互联网改变生活
- 北漂、南漂成为现象
- 房价第一次引起全民关注
- 新旧价值观激烈碰撞

---

### 顺叙线时代背景（2009-2025）

#### 2009-2012：金融危机后的调整期

- 2008年金融危机余波
- 北京奥运后的平静期
- 房价第一轮暴涨
- 80后开始结婚生子
- 北漂压力加剧

**社会氛围：**
- 结婚的经济压力巨大
- 房子成为婚姻必需品
- 城市生活成本快速上升
- 回二三线城市的声音出现

#### 2012-2015：移动互联网爆发

- 智能手机普及
- 微信、微博改变生活方式
- O2O、电商大战
- 程序员收入达到顶峰
- 北上深房价遥不可及

**社会氛围：**
- "风口"、"创业"成为热词
- 互联网公司快速扩张
- 加班文化盛行
- 一线城市生活节奏加快

#### 2015-2018：新经济与回归潮

- 共享经济、人工智能兴起
- 一线城市生活成本飙升
- 二三线城市"抢人大战"
- 逃离北上广成为话题
- 中年危机开始显现

**社会氛围：**
- 35岁危机被热议
- 回家乡成为选项
- 二线城市房价暴涨
- 生育率下降

#### 2018-2025：疫情与反思

- 2020-2022年：疫情冲击
- 远程办公常态化
- 互联网行业调整期（裁员、35岁危机）
- 回乡、回归成为趋势
- 对人生意义的重新思考

**社会氛围：**
- 疫情改变生活方式
- 互联网红利消退
- 降薪、裁员成为常态
- 躺平、内卷成为热词
- 重新审视人生价值

---

## 【使用建议】

### 在倒叙线中：
- 用具体事件标记时间（如"包产到户那年"）
- 通过物件变化展现时代（电视机、自行车）
- 用人物命运折射大时代（下海、打工、考学）

### 在顺叙线中：
- 用技术迭代标记时间（智能手机、移动支付）
- 通过房价、物价展现变化
- 用职业发展折射时代（互联网浪潮、中年危机）

### 时间锚点示例：

```
✓ "那年粮站刚塌，我出生。后来听说是包产到户第二年。"
✓ "我结婚那年，北京奥运刚结束，金融危机的余波还在。"
✓ "iPhone 4刚出来，我在北京五环外的出租屋里写代码。"
✓ "疫情那三年，我们在郑州买了房。"
```
