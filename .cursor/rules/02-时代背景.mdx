# 02 - 时代背景

## 【基本信息】

- **时间跨度**：1979-2025（46年）
- **主角**：石头，1979年10月14日出生，射手座
- **地理跨度**：河南洛阳伏牛山下→深圳（1997）→北京（2009）→上海（2014）→深圳（2016）→郑州（2021）
- **核心地域**：
  - 倒叙线：石洼村（虚构），伊河故道，伏牛山区
  - 顺叙线：深圳、北京、上海、郑州
- **职业轨迹**：
  - 1996-1997：在家务工
  - 1997-2008：深圳打工（小工→班组长→车间管理→创业开厂）
  - 2008：29岁结婚，成为小老板
  - 2009-2014：北京发展
  - 2014-2016：上海
  - 2016-2021：深圳
  - 2021至今：郑州定居

---

## 【历史事件背景】

历史事件作为时间锚点，帮助定位故事发生的具体时代背景。

### 倒叙线时代背景（1979-2009）

#### 1978-1982：改革开放初期

- 1978年：十一届三中全会，改革开放启动
- 1979-1980：知青返城，上山下乡结束
- 1980-1982：包产到户，家庭联产承包责任制
- 集体经济解体，粮站、供销社的衰落
- 村里开始有"万元户"

**社会氛围：**
- 从集体到个人的转变
- 人们对未来充满期待和不安
- 传统秩序开始松动

#### 1983-1992：乡村巨变期

- 计划生育政策严格执行
- 外出打工潮开始（山西煤矿、广东工厂）
- 电视机、自行车、缝纫机进入农村
- 乡镇企业兴起又衰落
- 读书改变命运的信念达到顶峰

**社会氛围：**
- 乡村开始空心化
- 留守老人和儿童出现
- 城乡差距初现
- 教育热潮

#### 1993-1999：城乡差距扩大

- 1992年：邓小平南巡讲话
- 市场经济体制确立
- 农民工大潮涌入城市（**1997年，18岁的我去了深圳**）
- 教育产业化，农村孩子上学越来越贵
- 乡村空心化加剧，留守老人和儿童
- **1996年，我高二辍学，在家务工一年**

**社会氛围：**
- 下海经商热潮
- "万元户"变成"老板"
- 农村与城市的鸿沟拉大
- 读书不一定改变命运的声音出现
- **深圳：制造业腾飞，遍地是工厂和机会**

#### 2000-2009：制造业黄金期与互联网萌芽

- 2000年左右：互联网开始普及
- **深圳制造业黄金期**（我从小工慢慢成长）
- 北上广深房价开始飙升
- 农村土地被征用，拆迁户出现
- 城乡二元结构固化
- **2008年，我29岁结婚，在深圳开厂，有了几个员工**
- **2009年，我30岁，去北京发展**

**社会氛围：**
- 制造业繁荣，珠三角是世界工厂
- 农民工变老板的故事开始出现
- 北漂、南漂成为现象
- 房价第一次引起全民关注
- 新旧价值观激烈碰撞

---

### 顺叙线时代背景（2009-2025）

#### 2009-2012：金融危机后的调整期

- 2008年金融危机余波
- 北京奥运后的平静期
- 房价第一轮暴涨
- 80后开始结婚生子
- 北漂压力加剧
- **我30-33岁，在北京发展，已婚，制造业生意**

**社会氛围：**
- 结婚的经济压力巨大
- 房子成为婚姻必需品
- 城市生活成本快速上升
- 回二三线城市的声音出现
- 制造业开始面临转型压力

#### 2012-2015：移动互联网爆发与制造业转型

- 智能手机普及
- 微信、微博改变生活方式
- O2O、电商大战
- 北上深房价遥不可及
- **制造业面临转型升级压力**
- **2014年，我35岁，去上海发展**

**社会氛围：**
- "风口"、"创业"成为热词
- 互联网公司快速扩张
- 传统制造业老板也在摸索转型
- 一线城市生活节奏加快
- 生意越来越难做

#### 2015-2018：新经济与回归潮

- 共享经济、人工智能兴起
- 一线城市生活成本飙升
- 二三线城市"抢人大战"
- 逃离北上广成为话题
- 中年危机开始显现
- **2016年，我37岁，离开上海回到深圳**
- **制造业进入困难期，环保压力、人工成本上升**

**社会氛围：**
- 35岁危机被热议（**我刚过35岁**）
- 回家乡成为选项
- 二线城市房价暴涨
- 生育率下降
- 实体经济不如从前

#### 2018-2025：疫情与反思

- 2020-2022年：疫情冲击
- 远程办公常态化
- 制造业遭受重创（供应链断裂、订单减少）
- 回乡、回归成为趋势
- 对人生意义的重新思考
- **2021年，我42岁，从深圳回到郑州定居**
- **制造业艰难求生，小老板的困境**

**社会氛围：**
- 疫情改变生活方式
- 实体经济艰难
- 降薪、裁员成为常态
- 躺平、内卷成为热词
- 重新审视人生价值
- **制造业老板的中年困境：成本上升、利润下降、不知未来在哪**

## 【使用建议】

### 在倒叙线中：
- 用具体事件标记时间（如"包产到户那年"）
- 通过物件变化展现时代（电视机、自行车）
- 用人物命运折射大时代（下海、打工、考学）

### 在顺叙线中：
- 用技术迭代标记时间（智能手机、移动支付）
- 通过房价、物价展现变化
- 用职业发展折射时代（互联网浪潮、中年危机）

### 时间锚点示例：

```
✓ "那年粮站刚塌，我出生。后来听说是包产到户第二年。"
✓ "我结婚那年，北京奥运刚结束，金融危机的余波还在。"
✓ "iPhone 4刚出来，我在北京五环外的出租屋里写代码。"
✓ "疫情那三年，我们在郑州买了房。"
```
