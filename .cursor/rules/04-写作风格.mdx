# 04 - 写作风格

## 【最高原则：冷静叙事】

这是整部作品的基调，所有技法都要服务于此。

### 少用修饰性语言

- 避免形容词堆砌
- 避免比喻过于华丽
- 避免排比、对仗等修辞
- 用最简单的动词和名词

### 克制的语调

- 像在陈述事实，不是在讲故事
- 像在做口述历史，不是在抒情
- 越是重大事件，越要平静地说出来
- 越是悲惨命运，越不能渲染

### 示例对比

```
✗ "母亲心如刀绞，泪如雨下，她撕心裂肺地哭喊着，仿佛整个世界都崩塌了。"

✓ "母亲坐在炕沿上，手里还拿着针线。她看着门外很久，没说话。"
```

### 为什么要冷静叙事？

- 因为真正的痛苦说不出来
- 因为巨大的苦难需要距离感
- 因为克制比渲染更有力量
- 因为读者的想象比作者的描述更震撼

---

## 【情感暗流：不直接抒情】

这是本作品的灵魂技法，最难掌握。

### 通过"不写"来写

- 不写人物的内心活动，只写外在表现
- 不写"他很悲伤"，而写他做了什么
- 不写"她很绝望"，而写她的沉默和动作
- 让情感在字里行间流动，不要说破

### 四种制造张力的技法

#### 1. 用命运的无常制造张力
- 前一刻还在说笑，下一刻就出事了
- 不解释为什么，只呈现结果
- 让读者自己感受命运的残酷

#### 2. 用人物的沉默制造张力
- 在应该说话的时候沉默
- 沉默比千言万语更有力量
- 沉默中包含了所有无法言说的东西

#### 3. 用行为冲突制造张力
- 父亲打骂孩子，但偷偷给他留了钱
- 母亲嘴上骂人，但行动上在帮助
- 爱与恨纠缠，不要说破

#### 4. 用细节的反差制造张力
- 在悲惨的事件后写一个温暖的细节
- 在温馨的场景里埋下不安的伏笔
- 让读者在情感的起伏中感受复杂性

### 示例

```
✗ "父亲其实很爱我，只是不善于表达。他的心里充满了愧疚和痛苦。"

✓ "父亲转身出门。第二天早上，我枕头底下多了二十块钱。"
```

---

## 【对话的"不说破"】

- 保持对话的模糊性和留白
- 让人物说话时欲言又止
- 用沉默、叹气、重复来传达情绪
- 最重要的话往往说不出口

### 示例

```
✓ "她是不是真的和王屠户好过?"
   三叔沉默了很久："谁知道呢?就算好过,又怎么样?"

✓ "你什么时候回郑州？"妻子在电话里问。
   "不知道。"我盯着深圳湾的夜景，"再等等吧。"

✓ "爹，我考上了。"
   父亲"嗯"了一声，低头继续抽烟。
```

---

## 【残酷的诗意】

- 用美的语言写残酷的事
- 将暴力、死亡、贫困写出诗意
- 城市的孤独也可以很诗意

### 示例

```
✓ "河床裂开的口子里,去年淹死的货郎留下的扁担还插在淤泥里,上面结满了冰。"

✓ "凌晨三点，望京SOHO的灯还亮着一半。程序员们在格子间里修复bug，像在修复自己的人生。"
```

---

## 【白描手法】

- **要**：不动声色的描写，比煽情更有力量
- **要**：用细节和动作展现情感，不要直接说出来
- **不要**：过度抒情和议论，中年视角更需要克制

### 示例

```
✓ "孙寡妇拿着信坐在窑檐下,从早上坐到太阳落山。"
✗ "孙寡妇心如刀绞，悲痛欲绝。"

✓ "我在北京五环外的出租屋里写了第一千行代码，窗外是雾霾。"
✗ "北京的生活让我感到无比焦虑和迷茫。"
```

---

## 【方言与口语化】

### 倒叙线（乡村）：适度使用河南方言
- "俺"、"咋"、"恁"、"啥"、"中不中"
- 对话中多用，叙述中少用

### 顺叙线（城市）：普通话为主，偶尔方言
- 在城市说方言时的尴尬
- 在家乡说普通话时的疏离
- 方言是身份标记，要谨慎使用

---

## 【程序员语言特色】（顺叙线专属）

- 用技术隐喻表达情感：
  ```
  ✓ "婚姻就像一段遗留代码，你接手时以为能重构，后来发现只能打补丁。"
  ✓ "我试图debug我的人生，但总是找不到那个关键的异常。"
  ```
- 不要过度使用技术术语，保持可读性
- 技术细节要真实，但不要炫技

---

## 【核心三句话】

1. **写什么人们做了什么，不写人们感受到什么**
2. **让读者哭，不要让人物哭**
3. **越是残酷的事，越要用平静的语言**
