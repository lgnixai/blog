# 02 - 叙事引擎

## 【章节OKR】

- 目标：本章要推进哪条线（个人/地理/时代/情感）？
- 张力：制造何种冲突（外部/内部/时代）？
- 伏笔：埋下哪枚意象或信息缺口？
- 兑现：本章兑现/部分兑现/延迟兑现？

### 自检清单
- [ ] 开篇是否有可感的钩子（10-80字）
- [ ] 至少一次不可逆的变化（代价/损失/选择）
- [ ] 一处回声（呼应既有意象/台词/动作）
- [ ] 时间标注清晰（年份/年龄/地点）

---

## 【引擎组件】

1. 钩子 Hook：未完成的动作/问题/画面。
2. 任务 Problem：角色此刻必须解决的具体事。
3. 行动 Action：推进事件的外在行为。
4. 转折 Turn：信息反转/力量对比逆转。
5. 代价 Cost：为选择付账（尊严/关系/健康/金钱）。
6. 沉默 Silence：不说破，让空白承载情绪。
7. 回声 Echo：意象/台词/动作的再次出现但意义改变。
8. 兑现 Payoff：对悬念的部分或全部回答。

### 极短示例
```
✓ 钩子：结婚照掉在地上，玻璃碎了一角。
→ 任务：今天得把证件补好。
→ 转折：翻抽屉时摸到那三枚铜钱。
→ 代价：迟到，和领导起冲突。
→ 回声：晚上回家，儿子拿起那枚最花的铜钱问：这是谁给你的？
```

---

## 【双时间线齿轮】

- 交汇点：2009年，结婚。
- 每章A/B双齿轮：A（倒叙1979-2009）⇄ B（顺叙2009-2025）。
- 切换触发器：物件/气味/声音/场景/对话。
- 回声位：每次切换后，安排一个细节呼应前一线的意象或动作。

### 切换范式
```
【顺叙】2018，郑州。父亲说：“这房子比窑洞强多了。”
---
【倒叙】1985，石洼村。第一次听说“楼房”。
```

---

## 【节奏曲线】

- 句法：信息密度高处用短句；留白前后放慢。
- 段落：3-5句一个呼吸；对话与白描交替。
- 章内：起（钩子）—承（行动）—转（代价）—合（回声）。
- 篇级：每3章一个小高潮；每篇一个大命题的阶段性回答。

---

## 【微悬念库】

- 提问型：一句未回答的问题（谁把扁担搬走了？）
- 动作中断型：门刚推开一条缝，里面的灯亮着……
- 意象错位型：夏天的河里飘着碎冰一样的塑料。
- 讯息缺口型：他把话说到一半，咽了回去。

---

## 【兑现原则】

- 延迟与长度匹配：小悬念≤1章；中悬念≤3章；大悬念留到篇尾。
- 兑现优先“情理之中，意料之外”。
- 兑现时重“代价”，轻“解释”。

### 兑现示例
```
伏笔（倒叙）：顺子袖口里那根红头绳。
兑现（多年后）：清理粮站废墟时，从他衣襟里抖出一截褪色的红绸。
```

---

## 【工具化输出】

- 章节卡：时间｜地点｜年龄｜目标｜冲突｜代价｜回声｜意象
- 切换卡：触发器｜上一线回声位｜下一线钩子
- 意象卡：首次出现｜变体1｜反转｜升华｜禁用法
