# 09 - 章节蓝图（可直接套用）

## 【蓝图0】通用双线章

```
【顺叙】YYYY，城市，我XX岁（钩子：一个动作未完成）
（场景与对话推进：行动—转折—代价—回声位）
---
【倒叙】YYYY，石洼村，我XX岁（触发器：物件/气味/声音）
（场景与对话推进：行动—转折—代价—回声位）
---
【顺叙】回到当下，用一个意象收束，不说破
```

## 【蓝图1】人物特写章（全知单线）

```
开场：人物在做一件具体的事
中段：两个时间节点的对照（不必标注年份）
结尾：一个沉默/一个动作，留白
```

## 【蓝图2】事件冲击章（高张力）

```
钩子：突发事件（塌方/火灾/裁员）
推进：三段动作，每段只一件事
代价：清晰可见，可计量
收束：回声位，指向人物弧线
```

## 【蓝图3】返乡对照章（空间对比）

```
城市开场：地铁/写字楼/物业大厅
触发：旧物/气味
乡村段落：窑洞/打谷场/河岸
合拢：在城市完成一个类似动作（对照）
```

---

## 【蓝图用法】

- 在《02-叙事引擎》确定章OKR后，再选择蓝图。
- 保持每章“一钩子一代价一回声”。
- 用《07-意象与符号》填充回声位。
