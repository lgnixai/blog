# 11 - 章节规范

## 【文件组织】

### 主线章节
- 文件名：`1.md`, `2.md`, `3.md`...
- 每个文件 3000-5000 字为宜
- 按照双时间线交替推进

### 人物特写章节
- 目录结构：`人物名/章节号.md`
- 例如：`秀梅/2.md`、`林晓/1.md`
- 用于"我"无法在场的重要场景
- 采用全知视角

### 章节编号建议
```
主线：1.md, 2.md, 3.md...
人物特写：人物名/数字.md
番外：extra/xxx.md
```

---

## 【章节结构】

### 一、开场（200-500字）

#### 格式：
```
【倒叙/顺叙】年份，地点，我X岁

第一段：一个场景或意象
```

#### 示例：
```
【倒叙】1983年，石洼村，我刚出生

伏牛山下了整整一夜的雪。

伊河水从秋天起就断流了。河床裂开的口子里...
```

### 二、主体（2500-4000字）

- **2-3条情节线交织**
- 倒叙线：童年记忆碎片
- 顺叙线：当下生活片段
- 可以在章内切换时间线

### 三、结尾（200-500字）

- **留白或回到意象**
- 不要总结性的议论
- 可以埋下伏笔

#### 示例：
```
我站在窑洞门口，手里还攥着那三枚铜钱。

锈迹斑斑。
```

---

## 【章节命名建议】

### 用具象物件：
- 《粮站》《铜钱》《扁担》《头绳》《核桃》
- 《钢笔》《电脑》《机票》《房产证》

### 用时间节点：
- 《一九八三年腊月》《二〇〇九年夏天》
- 《结婚那年》《疫情三年》

### 用人物名：
- 《孙寡妇》《林晓》《王屠户》
- 《父亲》《母亲》《三叔》

### 用地点：
- 《石洼村》《伊河》《窑洞》
- 《五环外》《望京》《深圳湾》

---

## 【章节节奏】

### 前期（第一篇：《离家》）
- 倒叙和顺叙各占一半
- 建立双时间线的基本框架
- 引出主要人物和冲突

### 中期（第二篇：《浮沉》）
- 双线开始频繁交叉
- 真相逐渐揭露
- 矛盾逐渐激化

### 后期（第三篇：《归处》）
- 双线逐渐靠拢
- 最后几章合二为一
- 达到情感的高潮和释放

---

## 【写作流程】

### 新建章节前：
1. 确定时间（哪一年）
2. 确定视角（"我"还是全知）
3. 确定核心事件
4. 确定要推进哪条线索

### 写作时：
1. 先写场景和对话，再补充心理
2. 多用短句，避免长句套长句
3. 每隔几段要有一个意象或细节
4. 不要连续三段都是叙述，要穿插对话

### 修改时检查：
- [ ] 是否有过度说明？
- [ ] 是否有情感泛滥？
- [ ] 是否保持了克制？
- [ ] 意象是否自然出现？
- [ ] 人物对话是否符合身份？
- [ ] 是否有让读者停下来思考的地方？

---

## 【字数控制】

### 单章字数：
- 最少：3000字
- 适中：3500-4500字
- 最多：5000字
- 超过5000字考虑拆分

### 三部曲字数预估：
- 第一篇：10-12章，约40000-50000字
- 第二篇：10-12章，约40000-50000字
- 第三篇：8-10章，约35000-45000字
- **总计**：约12-15万字

---

## 【章节内时间线切换】

### 切换次数：
- 少：1次（整章单线）
- 适中：2-3次（双线交织）
- 多：4-5次（关键章节）

### 切换位置：
- 章节开头和结尾各一次
- 中间根据需要切换
- 用`---`分隔

### 切换标记：
```
【顺叙】2018年，郑州，我39岁

（正文...）

---

【倒叙】1985年，石洼村，我6岁

（正文...）

---

【顺叙】（回到2018年）

（正文...）
```
