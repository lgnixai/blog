## 🚀 三种使用场景

### 场景1：第一次使用（建议用时2小时）

#### Step 1：建立整体认知（30分钟）
```
1. 阅读 README.mdx（10分钟）
   - 了解系统框架
   - 明确核心目标

2. 阅读 00-总纲.mdx（20分钟）
   - 理解三大支柱
   - 掌握核心原则
```

#### Step 2：掌握核心理论（90分钟）
```
3. 阅读 01-叙事哲学.mdx（30分钟）
   - 冰山理论
   - 沉默叙事
   - 残酷诗意

4. 阅读 02-人物宇宙.mdx（30分钟）
   - 三代人图谱
   - 核心人物
   - 群像原则

5. 阅读 03-时空结构.mdx（30分钟）
   - 双时间线
   - 交织技法
   - 章节架构
```

#### 完成后你将：
✅ 理解作品的核心DNA  
✅ 掌握基本创作方法  
✅ 可以开始尝试写作

### 场景2：开始新章节前（10分钟检查）

#### 快速检查清单

**1. 时间线确认**（2分钟）
```
□ 这是倒叙还是顺叙？
□ 具体年份？主角年龄？
□ 对应什么历史事件？

→ 查阅：03-时空结构.mdx 第X节
```

**2. 人物确认**（2分钟）
```
□ 本章涉及哪些人物？
□ 人物性格是否一致？
□ 人物关系是否准确？

→ 查阅：02-人物宇宙.mdx 对应人物
```

**3. 细节准备**（3分钟）
```
□ 倒叙线需要什么乡土细节？
  → 05-乡土密码.mdx

□ 顺叙线在哪座城市？
  → 06-城市图鉴.mdx 对应城市

□ 需要什么意象呼应？
  → 08-意象网络.mdx 对应意象
```

**4. 风格确认**（3分钟）
```
□ 叙事是否克制？
  → 01-叙事哲学.mdx 检查

□ 语言是否简洁？
  → 04-语言美学.mdx 检查

□ 方言使用是否得当？
  → 07-方言系统.mdx 检查
```

### 场景3：修改打磨时（15分钟检查）

#### 第一轮：内容检查（5分钟）

**禁忌检查**
```
□ 是否煽情了？
□ 是否说教了？
□ 是否脸谱化了？
□ 是否过度象征了？

→ 对照：12-创作禁忌.mdx 全部20条

**真实性检查**
```
□ 地理细节准确吗？
□ 历史事件对应吗？
□ 方言使用自然吗？
□ 有编造的风俗吗？

→ 对照：05-乡土密码.mdx / 07-方言系统.mdx

```

```

**对标茅奖标准**

## 💡 实战建议

### 第一次写作（建议写短章节试手）

**选题**：
- 选一个核心场景（如"父亲的二十块钱"）
- 500-1000字即可

**流程**：
1. 确认时间线（查03）
2. 准备细节（查05或06）
3. 设计意象（查08）
4. 开始写作（参照01、04）
5. 检查禁忌（查12）
6. 打磨语言（参照04）

**预期**：
- 用时：2-3小时
- 收获：熟悉系统
- 发现：自己的薄弱环节

### 进入创作状态后

**每天开始前**：
- 5分钟回顾核心原则
- 明确今天写什么（时间线、人物）

**写作中**：
- 不要频繁查阅（打断思路）
- 遇到问题做标记，写完再查

**写完后**：
- 必走检查流程（15分钟）
- 对照禁忌清单
- 打磨金句

## 📞 遇到问题怎么办？

### 问题类型 → 解决方案

**时间线混乱**
→ 建立时间表（Excel），标注每个事件

**人物性格不一致**
→ 建立人物卡片，记录关键特征

**方言使用不当**
→ 找河南老乡审读

**缺乏金句**
→ 每天摘抄10-传播设计.mdx的金句，培养语感

**写不下去了**
→ 重读01-叙事哲学.mdx，回归初心
