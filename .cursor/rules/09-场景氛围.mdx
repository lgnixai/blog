# 09 - 场景氛围

## 【环境描写的功能】

环境不是装饰，要有具体作用。

### 倒叙线（乡村）：
- **暗示人物命运**：雪、河水、塌方
- **烘托情绪**：窑洞的压抑、山坡的荒凉
- **推进情节**：火灾、水灾、粮站塌方
- **时代变迁**：粮站荒废、拖拉机进村、电视机

### 顺叙线（城市）：
- **暗示身份困境**：出租屋的狭窄、写字楼的冰冷
- **烘托孤独感**：地铁、夜班、空荡荡的街道
- **推进情节**：换城市、换工作、买房
- **时代标记**：iPhone、共享单车、疫情口罩

---

## 【感官细节】

### 倒叙线：

#### 视觉
- 黄土：赭色的、干裂的
- 血：王屠户院里的血迹
- 灰尘：铝矾土矿的灰尘
- 煤油灯：昏黄的光
- 雪：覆盖一切的白

#### 听觉
- 风声：呼呼的，穿过窑洞
- 哭声：孙寡妇的、婴儿的
- 窑洞土屑掉落：簌簌的
- 打谷声：啪啪的
- 蝉鸣：夏天的

#### 嗅觉
- 血腥味：王屠户家的
- 泥土味：雨后的
- 煤油味：灯里的
- 汗味：夏天干活的
- 饭香：玉米糊、烩面

#### 触觉
- 粗糙的手：父亲的、母亲的
- 冻裂的土地：冬天的
- 扎人的麦茬：收割后的
- 冰冷的河水：伊河的

---

### 顺叙线：

#### 视觉
- 电脑屏幕的蓝光：深夜加班
- 霓虹灯：城市的夜
- 雾霾：北京的天
- 高楼：密密麻麻的
- 出租屋：昏暗的、狭小的

#### 听觉
- 键盘声：啪啪啪的
- 地铁报站：机械的
- 外卖电话铃：刺耳的
- 汽车鸣笛：不间断的
- 空调噪音：嗡嗡的

#### 嗅觉
- 咖啡：公司里的
- 外卖：盒饭的味道
- 空调的味道：陈旧的
- 地铁里的味道：混杂的
- 高铁上的味道：消毒水

#### 触觉
- 鼠标：光滑的、冰冷的
- 手机屏幕：温热的
- 地铁扶手：油腻的
- 键盘：敲击的反馈
- 出租屋的床：潮湿的

---

## 【时间感的营造】

### 倒叙线：
- **农历节日**：腊月二十三、清明、中秋
- **季节和气候**：麦收、雪、伏天
- **人物的年龄增长**：换牙、长高、声音变粗
- **物件的变化**：电视机、自行车、拖拉机

### 顺叙线：
- **公历和项目节点**：春节回家、季度考核、年终奖
- **技术迭代**：从PC到移动互联网到AI
- **房价、物价的变化**：越来越贵
- **孩子的成长**：上学、学特长

---

## 【城市对比】（顺叙线专属）

### 北京（2009-2012）：
- **特征**：雾霾、地铁、五环外的出租屋
- **氛围**：梦想与焦虑并存
- **关键词**：买不起房的绝望
- **意象**：地铁里的人潮、望京的灯

### 上海（2012-2015）：
- **特征**：精致、冷漠、规则
- **氛围**：外来者的边缘感
- **关键词**：职业的上升期
- **意象**：外滩的灯光、地铁的冷气

### 深圳（2015-2018）：
- **特征**：年轻、快节奏、高收入
- **氛围**：职业巅峰
- **关键词**：离家越来越远
- **意象**：深圳湾的夜景、加班的灯火

### 郑州（2018-2025）：
- **特征**：回归、安顿、平淡
- **氛围**：离父母近了
- **关键词**：失去了什么？
- **意象**：黄河、烩面馆、新小区

## 【场景类型】

### 静态场景（描写为主）：
- 窑洞内部
- 出租屋清晨
- 打谷场日落
- 办公室深夜

### 动态场景（行动为主）：
- 粮站塌方
- 大火救人
- 搬家
- 春节回乡

### 对话场景：
- 父子对话
- 婆媳冲突
- 同事闲聊
- 夫妻争执

---

## 【使用原则】

### 1. 少即是多
```
✗ "窑洞里很暗，墙壁是土坯的，地上铺着黄土，
   有一张破旧的木床，一个小桌子，一盏煤油灯。"

✓ "窑洞里点着煤油灯。墙上的土簌簌往下掉。"
```

### 2. 环境要服务于人物
```
✓ "我站在北京五环外的天桥上，看着下面的车流。
   已经是第三次换出租屋了，越搬越远。"
```

### 3. 避免旅游指南式描写
```
✗ "郑州是河南省会，位于黄河南岸，有着悠久的历史..."

✓ "郑州的烩面馆，味道和小时候不一样了。"
```

### 4. 用环境制造对比
```
倒叙：窑洞里昏黄的煤油灯
顺叙：办公室里刺眼的荧光灯
```
